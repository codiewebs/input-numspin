Input numspin

A Pure javascript input spin plugin.

Demos

Basic

Add [data-numspin] attribute in input with type='number'.

<input type="number" class="form-control" data-numspin step="1" min="0" max="10" value="1">
Prefix

Set [data-prefix="**"] attribute with [data-numspin] input.

<input type="number" class="form-control" data-numspin data-prefix="$" step="1" min="0" max="15" value="1">

<input type="number" class="form-control" data-numspin data-prefix="%" step="1" min="0" max="15" value="1">
Suffix

Set [data-suffix="**"] attribute with [data-numspin] input.

<input type="number" class="form-control" data-numspin data-suffix="$" step="1" min="0" max="15" value="1">

<input type="number" class="form-control" data-numspin data-suffix="%" step="1" min="0" max="15" value="1">
Sizing

Set [data-input-size="sm"] attribute with [data-numspin] input form small element.

<input type="number" class="form-control" data-numspin data-input-size="sm" step="1" min="0" max="15" value="1">

<input type="number" class="form-control" data-numspin step="1" min="0" max="15" value="1">
Styles

Set [numspin-input-style="**"] attribute with [data-numspin] for different input styles.

<!-- Square input style example -->
<input type="number" class="form-control" data-numspin numspin-input-style="square" step="1" min="0" max="15" value="1">

<!-- Rounded input style example -->
<input type="number" class="form-control" data-numspin numspin-input-style="rounded" step="1" min="0" max="15" value="1">
Scheme

Set [numspin-input-scheme="**"] attribute with [data-numspin] for different input scheme.

<!-- Default (Solid) input scheme example -->
<input type="number" class="form-control" data-numspin numspin-input-scheme="solid" step="1" min="0" max="15" value="1">

<!-- flat input scheme example -->
<input type="number" class="form-control" data-numspin numspin-input-scheme="flat" step="1" min="0" max="15" value="1">
